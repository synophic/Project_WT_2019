ให้เขียน index.html ในโฟลเดอร์เดียวกับไฟล์นี้
แล้วเขียนข้อมูลของหน้าเว็บต่างๆในโฟลเดอร์ info โดยตั้งชื่อตามหน้าเช่น 1.html 2.html..... 22.html
เพื่อความสะดวกในการหานะไอสัส

ในโฟลเดอร์ info จะมีไฟล์ info.json เป็นไฟล์ที่เก็บข้อมูลหน้าต่างๆโดยจะมีรายละเอียดดังนี้
ถายใน array ใหญ่ จะมี object ต่างๆโดยเรียงตามหน้าเอาใว้แล้ว
โดยภายใน object นี้จะมี 2key หลักได้แก่
topic: ชื่อของหัวข้อหลัก แต่เนื่องจากเราทำเรื่องอยุธยาอย่างเดียวจึงเหมือนกันทุกตัว
Info: จะเป็น array ที่มี object ย่อยอีกที
โดย object ย่อยนั้นจะประกอบไปด้วย
title: ชื่อหัวข้อ
type: มี 2 แบบ text, multiple
data: จะจัดเรียงตาม type
-ประเภท text ข้อมูลจะเป็นstring
-ประเภท multiple จะมีarray ย่อยซ้อนไปอีกขั้น โดยจะมีarray 2 ประเภท
	-ประเภทแรกจะเก็บค่า string ตัวเดียวเท่านั้น index 0
	-ประเภทที่2จะเก็บค่า2ตัว โดย
		-index 0 เก็บค่า string
		-index 1 เก็บค่า stringหลายตัวเป็น array(หากนำมาใช้แนะนำให้ใช้ tag unorder list)

โฟลเดอร์ style เอาใว้เก็บ css js เรื่องของมึงเลย

โฟลเดอร์ image ใว้เก็บภาพ

ถ้าไฟล์ json ดึงข้อมูลตัวmultipleที่มี index 2 ตัวไม่ได้ ก็ลองเปลี่ยนดู ถ้าทำไมได้ก็ตายห่าไปซะ แค่นี่ละ
